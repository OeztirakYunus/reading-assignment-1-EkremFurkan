Cheat Sheet

Word Wide Web General

1).What are the most essential technologies that made the World Wide Web such a success?
1.) A system of globally unique identifiers for resources on the Web and elsewhere, the universal document identifier (UDI), later known as uniform resource locator (URL) and uniform resource identifier (URI).
2.) The publishing language HyperText Markup Language (HTML).
3.) The Hypertext Transfer Protocol (HTTP).

2.) How are resources (html pages, images, etc) addressed in the web?
A: As URI and URL.

3.) What are the general components of a url?
A: A protocol (http), a hostname (www.example.com), and a file name (index.html).

Web Server

1.) What is a web server?
A: A Server that transfers Web pages to clients

2.) What does a web server?
A: It stores, processes and transfers Web pages to clients. The communication between client and server takes place using the Hypertext Transfer Protocol (HTTP).

3.) How can content be stored on a web server?
A: Through communication between client and server through using the Hypertext Transfer Protocol (HTTP).

4.) How is content in webpages described?
A: As the stuff in your Web site.
This may include documents, data, applications, e-services, images, audio and video files, personal Web pages, archived e-mail messages, and more.

Web Client

1.) What is a web client?
A: It is a software application for accessing information on the World Wide Web

2.) What does a web client?
A: The purpose of a web browser is to fetch information resources and display them on a user's device.
This process begins when the user inputs a URL, such as https://en.wikipedia.org/, into the browser.
Virtually all URLs on the Web start with either http: or https: which means the browser will retrieve them with the Hypertext Transfer Protocol.

3.) Web Client and Web Browser is the same
A: Yes

Hello World

1.) What is the html head element?
A: The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

2.) What does mean?
A: It is an instruction to the web browser about what version of HTML the page is written in.

3.) What are the two top most elements within the html element?
A: Head and body.

4.) How to emphasize a text and how is it displayed in the browser?
A: With (em) and it is displayed as a emphasized text.

5.) How to mark up important ext and how is it displayed in the browser?
A: With (mark) and it is displayed as a marked text.
